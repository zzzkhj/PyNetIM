class pynetim.diffusion_model.base_diffusion_model.BaseDiffusionModel(graph: pynetim.graph.IMGraph, init_seeds: list, record_states: bool = False)[源代码]

扩散模型基类。

该类为各种扩散模型实现提供了基础框架，包含图结构、初始种子集、状态记录功能， 以及需要子类实现的核心方法。支持在网络图上模拟信息、疾病或其他 实体的传播过程。

重置模型状态。

将模型恢复到初始状态，如果启用状态记录则清空已记录的状态历史， 并重新设置初始种子节点状态。
