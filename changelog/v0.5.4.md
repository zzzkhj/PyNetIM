# v0.5.4 更新日志

**发布日期**: 2026-04-28

## 概述

v0.5.4 是一个架构优化版本，主要重构了算法继承结构和目录组织，新增了种群优化算法基类、强化学习算法模块，并修复了深度学习基类缺少扩散模型支持的问题。同时引入了四个新算法：RLSetGWOAlgorithm、SADPEAAlgorithm、CoreQAlgorithm 和 TCQAlgorithm。

## 🎯 新增功能

### 权重管理器模块

新增 `pynetim.weights` 模块，提供类似 Hugging Face Transformers 的预训练权重管理机制。

**核心功能**:

- 从远程 URL 自动下载预训练权重
- 本地缓存机制（`~/.pynetim/weights/`）
- 下载进度条显示（使用 tqdm）
- 权重文件校验

**支持的权重文件**:

| 权重文件                 | 算法        | 说明                |
| -------------------- | --------- | ----------------- |
| `bigdn_weights.pth`  | BiGDN     | Teacher 模型权重      |
| `bigdns_weights.pth` | BiGDN     | Student 模型权重（蒸馏后） |
| `node_encoder.pth`   | BiGDN     | 节点编码器权重           |
| `tripling.ckpt`      | ToupleGDD | Tripling 模型权重     |
| `s2vdqn.ckpt`        | S2V-DQN   | S2V-DQN 模型权重      |

**使用示例**:

```python
from pynetim.weights import WeightManager

# 获取权重路径（自动下载到缓存）
weights_path = WeightManager.get_weights_path("bigdn_weights.pth")
print(f"权重文件路径: {weights_path}")

# 查看缓存信息
cache_info = WeightManager.get_cache_info()
print(f"缓存目录: {cache_info['cache_dir']}")
print(f"文件数量: {cache_info['num_files']}")
print(f"总大小: {cache_info['total_size_mb']:.2f} MB")

# 列出缓存的权重
cached_weights = WeightManager.list_cached_weights()
for weight in cached_weights:
    print(f"  - {weight}")

# 清空缓存
WeightManager.clear_cache()
```

### 图分解函数

新增 `pynetim.graph.compute_k_shell_values` 函数，提供 K-shell 分解功能。

**算法**: Batagelj-Zaversnik 算法，时间复杂度 O(m)

**使用示例**:

```python
from pynetim import IMGraph
from pynetim.graph import compute_k_shell_values

graph = IMGraph(edges, directed=True)
k_shell = compute_k_shell_values(graph)
print(k_shell)  # {0: 1, 1: 2, 2: 1, ...}
```

**参考文献**: Batagelj, V., & Zaversnik, M. (2003). An O(m) algorithm for cores decomposition of networks. arXiv preprint cs/0310049.

### 随机图生成函数

新增 `pynetim.graph` 模块中的随机图生成函数，方便用户快速创建测试图。

**新增函数**:

| 函数                  | 模型              | 说明                      |
| ------------------- | --------------- | ----------------------- |

| `generate_ba_graph` | Barabási-Albert | 无标度网络，优先连接机制            |

```python
# WS 小世界网络
graph = generate_ws_graph(n=1000, k=4, p=0.1, random_seed=42)
```

### SIR 模型感染率计算

新增 `pynetim.utils.compute_sir_beta` 函数，基于配置模型近似计算 SIR 模型的感染率。

**公式**:

- β\_c = γ · <k> / \<k²>
- β = c · β\_c

**参数说明**:

- `<k>`: 平均度
- `<k²>`: 度的平方均值
- `γ`: 恢复率
- `c`: 调节系数（c<1 不爆发，c=1 临界，c>1 爆发）

**使用示例**:

```python
from pynetim.utils import generate_er_graph, compute_sir_beta

graph = generate_er_graph(n=100, p=0.1, random_seed=42)
beta, beta_c = compute_sir_beta(graph, gamma=0.1, c=1.3)
print(f"感染率: {beta:.4f}, 临界感染率: {beta_c:.4f}")
```

**参考文献**: Pastor-Satorras, R., Castellano, C., Van Mieghem, P., & Vespignani, A. (2015). Epidemic processes in complex networks. Reviews of Modern Physics, 87(3), 925.

### 边权重设置函数

新增 `pynetim.graph` 模块中的边权重设置函数，支持多种权重设置方式。

**新增函数**:

| 函数                      | 说明                     |
| ----------------------- | ---------------------- |
| `set_edge_weights`      | 通用边权重设置函数              |
| `set_const_weights`     | 设置常数边权重                |
| `set_tv_weights`        | 从列表中随机选择边权重            |
| `set_uniform_weights`   | 设置均匀分布边权重              |
| `set_wc_weights`        | 设置 WC 模型边权重（权重 = 1/入度） |
| `set_edge_weights_dict` | 通过字典设置边权重              |

**支持的权重类型**:

| 类型        | 说明                |
| --------- | ----------------- |
| `const`   | 常数权重              |
| `tv`      | 从列表中随机选择          |
| `uniform` | 均匀分布随机            |
| `wc`      | WC 模型，权重 = 1 / 入度 |

**使用示例**:

```python
from pynetim.graph import generate_er_graph, set_edge_weights, set_wc_weights

# 生成随机图
graph = generate_er_graph(n=100, p=0.1, random_seed=42)

# 通用函数设置权重
set_edge_weights(graph, "const", const_value=0.1)
set_edge_weights(graph, "wc")

# 直接调用具体函数
set_wc_weights(graph)

# 通过字典设置
from pynetim.graph import set_edge_weights_dict

weights = {(0, 1): 0.1, (1, 2): 0.2, (2, 3): 0.3}
set_edge_weights_dict(graph, weights)
```

### CoreQAlgorithm 算法

新增 K-core 层次引导的 Q-learning 影响力最大化算法 `CoreQAlgorithm`。

**参考文献**: Ahmad, W., & Wang, B. (2025). A learning-based influence maximization framework for complex networks via K-core hierarchies and reinforcement learning. Expert Systems With Applications, 259, 125393.

**使用示例**:

```python
from pynetim import IMGraph
from pynetim.algorithms.reinforcement_learning import CoreQAlgorithm

# 创建图
edges = [(0, 1, 0.1), (0, 2, 0.15), (1, 2, 0.2)]
graph = IMGraph(edges, directed=True)

# 初始化算法
algo = CoreQAlgorithm(
    graph,
    n_candidates=60,   # 候选集大小
    episodes=200,      # 训练轮数
    random_seed=42
)

# 运行算法
seeds = algo.run(k=10)
print(f"种子节点: {seeds}")
```

### TCQAlgorithm 算法

新增 K-core 引导的自适应 Q-learning 影响力最大化算法 `TCQAlgorithm`，支持目标约束 IM。

**参考文献**: Ahmad, W., & Wang, B. (2025). K-core-guided adaptive learning and policy optimization for targeted influence maximization in complex networks. Neurocomputing, 131612.

**使用示例**:

```python
from pynetim import IMGraph
from pynetim.algorithms.reinforcement_learning import TCQAlgorithm

# 创建图
edges = [(0, 1, 0.1), (0, 2, 0.15), (1, 2, 0.2)]
graph = IMGraph(edges, directed=True)

# 初始化算法
algo = TCQAlgorithm(
    graph,
    alpha=0.1,        # 学习率
    gamma=0.9,        # 折扣因子
    epsilon=0.5,      # 探索率
    n_candidates=6,   # 候选集倍数
    episodes=200,     # 训练轮数
    random_seed=42
)

# 运行算法
seeds = algo.run(k=10)
print(f"种子节点: {seeds}")
```

### RLSetGWOAlgorithm 算法

新增基于强化学习的灰狼优化影响力最大化算法 `RLSetGWOAlgorithm`。

**参考文献**: Roayaei, M. (2026). Using Reinforcement Learning to Boost Grey Wolf Optimizer for Influence Maximization in Social Networks. SN Computer Science, 7, 184.

### SADPEAAlgorithm 算法

新增结构感知双概率进化自适应算法 `SADPEAAlgorithm`，用于解决预算约束的影响力最大化问题。

**参考文献**: Zhu, E., Wang, H., Zhang, Y., & Ma, M. (2025). SADPEA: Structure-aware dual probability evolutionary adaptive algorithm for the budgeted influence maximization problem. Information Sciences, 699, 121784.

**使用示例**:

```python
from pynetim import IMGraph
from pynetim.algorithms.population import SADPEAAlgorithm

# 创建图
edges = [(0, 1, 0.1), (0, 2, 0.15), (1, 2, 0.2), (1, 3, 0.1)]
graph = IMGraph(edges, directed=True)

# 初始化算法
algo = SADPEAAlgorithm(
    graph,
    pop_size=20,          # 种群大小
    max_iter=50,          # 最大进化代数
    random_seed=42        # 随机种子
)

# 运行算法
seeds = algo.run(k=10)
print(f"种子节点: {seeds}")
```

### 传播模型新增 normalize 参数

所有传播模型的 `run_monte_carlo_diffusion` 和 `run_monte_carlo_with_frequency` 方法新增 `normalize` 参数，控制返回的平均影响范围是否归一化（除以图规模）。

**改进内容**:

- C++ 传播模型（IC、LT、SI、SIR）新增 `normalize` 参数
- Python 多进程传播模型新增 `normalize` 参数
- 默认 `normalize=False`，保持向后兼容

**使用示例**:

```python
from pynetim import IndependentCascadeModel

model = IndependentCascadeModel(graph, seeds={0, 1})

# 不归一化（默认），返回平均激活节点数
avg = model.run_monte_carlo_diffusion(1000)

# 归一化，返回平均激活比例（0~1）
avg_ratio = model.run_monte_carlo_diffusion(1000, normalize=True)
print(f"影响比例: {avg_ratio:.4f}")
```

### ToupleGDD 算法运行时错误修复

修复了 ToupleGDD 算法在不使用 DeepWalk 时，节点嵌入维度与模型输入不匹配导致的 `RuntimeError`。

**问题原因**: 当 `use_deepwalk=False` 时，节点嵌入维度为 `embed_dim * 2`，但代码中错误地使用了 `embed_dim`，导致 Tripling 模型前向传播时张量形状不匹配。

**修复内容**: 修正节点嵌入维度为 `embed_dim * 2`，与 DeepWalk 模式下的输出维度一致。

**新增功能**:

- 新增权重管理器模块 `pynetim.weights.WeightManager`

- 新增随机图生成函数 `generate_er_graph`, `generate_ba_graph`, `generate_ws_graph`
- 新增边权重设置函数 `set_edge_weights`, `set_const_weights`, `set_tv_weights`, `set_uniform_weights`, `set_wc_weights`, `set_edge_weights_dict`
- 新增 SIR 模型感染率计算函数 `compute_sir_beta`
- 新增强化学习算法模块 `pynetim.algorithms.reinforcement_learning`
- 新增 CoreQAlgorithm 算法（K-core 层次引导的 Q-learning）
- 新增 TCQAlgorithm 算法（K-core 引导的目标约束 Q-learning）
- 新增种群优化算法模块 `pynetim.algorithms.population`
- 新增 RLSetGWOAlgorithm 算法（基于强化学习的灰狼优化）
