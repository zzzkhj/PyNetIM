.. py:class:: IMGraph(graph: networkx.Graph | networkx.DiGraph, edge_weight_type: str, constant_weight: float = None)

   影响最大化图类，用于封装网络图结构和边权重设置。

   该类提供了对NetworkX图对象的封装，支持设置不同类型的边权重，
   并提供了便捷的图属性访问接口。

   .. attribute:: graph

      networkx图对象，可以是有向图或无向图

      :type: Graph | DiGraph

   .. attribute:: direction

      指示图是否有向

      :type: bool

   .. attribute:: number_of_nodes

      图中节点总数

      :type: int

   .. attribute:: number_of_edges

      图中边总数

      :type: int

   .. attribute:: edge_weight_type

      边权重类型

      :type: str


   .. py:attribute:: nx_graph


   .. py:attribute:: direction


   .. py:attribute:: number_of_nodes


   .. py:attribute:: number_of_edges


   .. py:attribute:: edge_weight_type


   .. py:property:: nodes

      获取图中所有节点。

      :returns: 图中所有节点的视图
      :rtype: NodeView


   .. py:property:: edges

      获取图中所有边。

      :returns: 图中所有边的视图
      :rtype: EdgeView


   .. py:method:: neighbors(node)

      获取指定节点的邻居节点。

      :param node: 节点标识符

      :returns: 指定节点的所有邻居节点迭代器
      :rtype: iterator



   .. py:method:: in_neighbors(node)

      获取指定节点的入邻居节点。

      对于有向图，返回前驱节点；对于无向图，返回所有邻居节点。

      :param node: 节点标识符

      :returns: 指定节点的入邻居节点迭代器
      :rtype: iterator



   .. py:method:: out_neighbors(node)

      获取指定节点的出邻居节点。（同neighbors方法）

      :param node: 节点标识符

      :returns: 指定节点的出邻居节点迭代器
      :rtype: iterator



   .. py:property:: in_degree

      获取图的入度。

      :returns: 图的入度视图
      :rtype: int


   .. py:property:: out_degree

      获取图的出度。

      :returns: 图的出度视图
      :rtype: int


   .. py:method:: degree()

      获取图的度。

      :returns: 图的度视图
      :rtype: int
