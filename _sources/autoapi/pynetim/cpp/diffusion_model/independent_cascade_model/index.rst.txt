.. py:class:: IndependentCascadeModel(seeds: Set[int], graph: pynetim.cpp.graph.graph.IMGraphCpp)

   Independent Cascade (IC) diffusion model.

   本类实现经典的 Independent Cascade 传播模型，
   用于模拟信息 / 影响力在图结构中的随机扩散过程。

   具体传播逻辑由 C++ 后端实现，
   本文件仅用于 Python 侧的类型提示与文档说明。


   .. py:method:: set_seeds(seeds: Set[int]) -> None

      更新种子节点集合

      :param seeds: 新的初始激活节点集合
      :type seeds: Set[int]



   .. py:method:: run_monte_carlo_diffusion(rounds: int, seed: int = ..., use_multithread: bool = ...) -> float

      运行 Monte Carlo 扩散模拟

      在给定初始种子节点的情况下，
      重复进行多轮随机扩散实验，
      统计最终被激活节点数的期望值。

      :param rounds: Monte Carlo 模拟轮数
      :type rounds: int
      :param seed: 随机数种子（用于保证实验可复现性）
      :type seed: int, optional
      :param use_multithread: 是否启用多线程加速模拟
      :type use_multithread: bool, optional

      :returns: 扩散结束后激活节点数量的期望值
      :rtype: float
