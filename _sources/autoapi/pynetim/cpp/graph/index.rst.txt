.. py:class:: IMGraphCpp(num_nodes: int, edges: List[Tuple[int, int]], weights: List[float] = ..., directed: bool = True)

   图结构类（C++ 后端实现，Python 类型存根）

   本类用于表示有向 / 无向加权图，主要服务于
   影响力传播（IC / LT / SIR 等）与相关算法。

   实际逻辑由 C++ 扩展模块（graph.pyd）实现，
   本文件仅用于类型提示与文档生成。
