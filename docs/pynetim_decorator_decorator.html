class Timer(builtins.object)
    Timer(repeat=1)
 
计时器装饰器类，用于测量函数执行时间。
 
该装饰器可以重复运行函数多次并计算平均执行时间。
 
Attributes:
    repeat (int): 函数每次调用时实际运行的次数
    total_time (float): 累计总执行时间
    call_count (int): 函数被调用的次数
