class DegreeDiscountAlgorithm(pynetim.algorithms.base_algorithm.BaseAlgorithm)
    DegreeDiscountAlgorithm(graph: pynetim.graph.graph.IMGraph, diffusion_model='IC')
 
度折扣启发式算法(Degree Discount)用于选择影响力最大化种子节点。
 
该算法是Single Discount的改进版本，考虑了邻居节点之间的影响关系，
使用更复杂的折扣公式来更好地评估节点的边际影响力。

class SingleDiscountAlgorithm(pynetim.algorithms.base_algorithm.BaseAlgorithm)
    SingleDiscountAlgorithm(graph: pynetim.graph.graph.IMGraph, diffusion_model=None)
 
简单度折扣启发式算法(Single Discount)用于选择影响力最大化种子节点。
 
该算法通过逐步选择具有最高度数的节点作为种子，并对其邻居节点的度数进行折扣，
以避免选择过多相互连接的节点。
